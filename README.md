# Promises-JS
# Promise Example in JavaScript

This project demonstrates how to use **JavaScript Promises** to handle asynchronous operations.

## Features
- Example tasks: 
  - Walk the dog  
  - Clean the kitchen  
  - Take out the trash  
- Each task simulates asynchronous work using `setTimeout`.
- Shows how to use:
  - `resolve` 
  - `reject` 
  - `.then()` chaining
  - `.catch()` error handling
